Configuración inicial · Conectar con Google Sheets
1Cree el Apps Script en Google Sheets
1. Abra Google Sheets → Extensiones → Apps Script
2. Borre el código existente y pegue el script de IDIMSA
3. Guarde (Ctrl+S) con nombre "IDIMSA"
4. Click en Implementar → Nueva implementación
5. Tipo: Aplicación web
6. Ejecutar como: Yo · Acceso: Cualquier persona
7. Copie la URL que termina en /exec
2Pegue la URL del Apps Script
⚠️ La URL debe terminar en /exec — no en /dev
